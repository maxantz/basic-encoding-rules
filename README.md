# basic-encoding-rules
This library implements the subset of ASN.1 BER as required by SMPTE ST 336 (KLV).

This software is based solely on publicly available international standards (ITU-T X.690 / ISO/IEC 8825-1 / SMPTE ST 336 / MISB-ST-0601.14).
It does not contain controlled technical data and is not subject to ITAR.
The software is classified as EAR99 unless otherwise incorporated into a controlled end product.
